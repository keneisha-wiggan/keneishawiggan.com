# Keneisha Wiggan Portfolio Website

This is a Vite + React + Tailwind CSS portfolio website package.

## Open in Visual Studio Code
1. Unzip this folder.
2. Open the folder in VS Code.
3. Open the terminal.
4. Run:

```bash
npm install
npm run dev
```

## Build for Hostinger
Run:

```bash
npm run build
```

This creates a `dist` folder.

## Upload to Hostinger
Upload the **contents inside `dist`** to Hostinger's `public_html` folder.

## Included public assets
- `public/kwiggan-headshot.jpg`
- `public/Keneisha_Wiggan_resume_top5_layout.pdf`

If you replace the headshot or resume, keep the same filenames or update the constants in `src/App.jsx`.
